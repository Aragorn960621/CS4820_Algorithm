{ \bf (2)} {\em (10 points)}
Call a Turing machine $M$ {\em termination-safe}
if $M(y) \neq \nearrow$ for all $y \in \Sigma^*$. In other words,
a termination-safe machine is one which is guaranteed to terminate
on every input string.

Let $T \subset \Sigma^*$ denote the set of all strings $x$
such that $x$ is the description of a termination-safe Turing
machine. Prove that $T$ is not recursively enumerable.
